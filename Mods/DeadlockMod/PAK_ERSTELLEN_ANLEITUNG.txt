⚙️ INSTALLATION & TESTING:
--------------------------

### Für Entwickler (Testing):
1. .pak Datei kopieren nach:
   C:\Program Files (x86)\Steam\steamapps\common\Deadlock\deadlock\addons\

2. Deadlock starten

3. Console öffnen (F1 oder ~) und eingeben:
   perf_debug  (aktiviert Debug-Modus)
   perf_demo   (aktiviert Demo-Modus falls Game Events nicht verfügbar)
   perf_toggle (zeigt HUD an)

### Für End-User:
1. .pak Datei in Deadlock\deadlock\addons\ Ordner kopieren
2. Deadlock starten  
3. Console (F1): perf_toggle
4. HUD erscheint rechts oben (Position anpassbar)

🎯 CONSOLE COMMANDS (VOLLSTÄNDIG):
---------------------------------
perf_toggle           # HUD ein/ausblenden
perf_reset           # Statistiken zurücksetzen
perf_position <x> <y> # HUD Position ändern (z.B. perf_position 100 50)
perf_debug           # Debug-Nachrichten ein/aus
perf_demo            # Demo-Modus mit simulierten Daten
perf_info            # Detaillierte Performance-Informationen anzeigen
