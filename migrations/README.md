# Migrations

Deadlock enthält keine Twitch-Migrationsskripte mehr. Twitch-bezogene Datenmigrationen, Analytics-Schemas und Betriebsanleitungen liegen jetzt im separaten Repo `Deadlock-Twitch-Bot`.

Für den verbliebenen `/streamer`-Flow in Deadlock werden keine lokalen Migrationsskripte mehr benötigt. Setup-Hinweise stehen in [../docs/TWITCH_EXTERNAL_REPO.md](../docs/TWITCH_EXTERNAL_REPO.md).
